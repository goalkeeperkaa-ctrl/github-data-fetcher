
# Aidagis 2.0 — Агрегатор мероприятий

## Обзор
Универсальная платформа для поиска и публикации мероприятий любого типа. Пользователи могут создавать мероприятия, искать их по фильтрам, просматривать на карте, сохранять в избранное и получать уведомления.

---

## Страницы и функции

### 1. Главная страница
- Лента мероприятий в виде карточек (название, дата, место, категория, изображение)
- Строка поиска и фильтры (по дате, категории, городу/району)
- Переключение между видом «список» и «карта»

### 2. Карта мероприятий
- Интерактивная карта с маркерами мероприятий
- При клике на маркер — превью мероприятия
- Фильтрация прямо на карте

### 3. Страница мероприятия (детали)
- Полное описание, дата/время, место, организатор
- Изображение/обложка мероприятия
- Кнопки: «Добавить в избранное», «Поделиться»
- Местоположение на мини-карте

### 4. Создание/редактирование мероприятия
- Форма: название, описание, дата/время, место (с выбором на карте), категория, загрузка изображения
- Доступно только авторизованным пользователям
- Возможность редактировать и удалять свои мероприятия

### 5. Регистрация и авторизация
- Вход и регистрация по email
- Профиль пользователя с аватаром и списком созданных мероприятий

### 6. Личный кабинет
- Мои мероприятия (созданные)
- Избранное (сохранённые мероприятия)
- Настройки уведомлений
- Редактирование профиля

### 7. Уведомления
- Напоминания о предстоящих мероприятиях из избранного
- Уведомления о новых мероприятиях в выбранных категориях

---

## Бэкенд (Lovable Cloud + Supabase)
- **База данных**: таблицы для мероприятий, пользователей, избранного, уведомлений
- **Аутентификация**: регистрация/вход через Supabase Auth
- **Хранилище**: загрузка изображений мероприятий через Supabase Storage
- **Безопасность**: RLS-политики для защиты данных пользователей
